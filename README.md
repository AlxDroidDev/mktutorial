# IO.Swagger - ASP.NET Core 2.0 Server

This is a simple API that is meant to be used only as an example

## Run

Linux/OS X:

```
sh build.sh
```

Windows:

```
build.bat
```

## Run in Docker

```
cd src/IO.Swagger
docker build -t io.swagger .
docker run -p 5000:5000 io.swagger
```
